# ECommerce_WebApp
# 🛒 ECommerce Web Application

A modern full-stack ECommerce web application built with **Next.js**, **TypeScript**, **Node.js**, **Express.js**, and **MySQL**. This project is being developed to demonstrate my full-stack development skills, scalable backend architecture, responsive UI design, secure authentication, and RESTful API development.

---

## 🚀 Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

### Backend
- Node.js
- Express.js
- TypeScript
- REST API

### Database
- MySQL
- mysql2

### Authentication
- JWT (JSON Web Token)
- bcrypt
- Google OAuth (Planned)
